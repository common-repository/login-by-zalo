== Installation ==

1. Upload the `login-by-zalo` directory to the `/wp-content/plugins/` directory via FTP
2. Alternatively, upload the `login-by-zalo.zip` file to the 'Plugins->Add New' page in your WordPress admin
area
3. Activate the plugin through the 'Plugins' menu in WordPress

1, 2, 3: You're done!
